# Структура шаблонов

Эта папка содержит все шаблоны для генерации Anki карточек.

## Структура файлов

### CSS

- **`styles.css`** - Стили для карточек Anki

### JavaScript

- **`tts_button.js`** - Функция для синтеза речи (Text-to-Speech)
- **`check_answer.js`** - Функции для проверки ответов пользователя

### HTML шаблоны карточек

#### Карточка 1: EN → RUS (Английское слово → Русский перевод)

- **`card_en_to_rus_front.html`** - Лицевая сторона (вопрос)
- **`card_en_to_rus_back.html`** - Обратная сторона (ответ)

#### Карточка 2: RUS → EN (Русский перевод → Английское слово)

- **`card_rus_to_en_front.html`** - Лицевая сторона (вопрос)
- **`card_rus_to_en_back.html`** - Обратная сторона (ответ)

#### Карточка 3: Example Practice (Практика с примерами)

- **`card_example_front.html`** - Лицевая сторона (вопрос)
- **`card_example_back.html`** - Обратная сторона (ответ)

## Переменные в шаблонах

В HTML шаблонах используются следующие переменные Anki:

- `{{Word}}` - Английское слово
- `{{Transcription}}` - Транскрипция
- `{{Translation}}` - Перевод на русский
- `{{ExampleEn}}` - Пример на английском
- `{{ExampleRu}}` - Пример на русском
- `{{AudioUrl}}` - URL аудио файла (опционально)

## Как редактировать

### Изменение стилей

1. Откройте `styles.css`
2. Измените нужные стили
3. Запустите `generate_words_deck.py` заново

### Изменение логики JavaScript

1. Откройте `tts_button.js` или `check_answer.js`
2. Измените нужные функции
3. Запустите `generate_words_deck.py` заново

### Изменение структуры карточек

1. Откройте нужный HTML файл (например, `card_en_to_rus_front.html`)
2. Измените разметку
3. Запустите `generate_words_deck.py` заново

## Примеры изменений

### Пример 1: Изменить цвет фона карточки

В `styles.css`:

```css
.card {
  background-color: #f0f0f0; /* вместо white */
}
```

### Пример 2: Изменить скорость речи TTS

В `tts_button.js`:

```javascript
utterance.rate = 0.6; // медленнее (было 0.8)
```

### Пример 3: Добавить подсказку

В `card_en_to_rus_front.html`:

```html
<div style="font-size: 14px; color: #999;">Подсказка: {{Translation}}</div>
```

## Технические детали

- Все файлы должны быть в кодировке UTF-8
- HTML шаблоны используют синтаксис Mustache ({{variable}})
- JavaScript встраивается в CSS через тег `<script>` (особенность Anki)
- CSS применяется ко всем карточкам одной модели

## Отладка

Если карточки не работают правильно:

1. Проверьте, что все файлы существуют
2. Проверьте кодировку файлов (должна быть UTF-8)
3. Проверьте консоль браузера в Anki (Tools → Preferences → Review → Show debug console)
4. Убедитесь, что переменные {{Variable}} написаны правильно
