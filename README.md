# Irregular English Verbs - Anki Deck Generator

Этот проект создает колоду карточек Anki для изучения неправильных английских глаголов с транскрипцией, переводом и примерами.

## Файлы проекта

- `generate_verbs_deck.py` - основной скрипт для генерации колоды Anki
- `verbs.csv` - база данных неправильных глаголов с транскрипциями и примерами
- `irregular_verbs.apkg` - готовая колода Anki (создается после выполнения скрипта)

## Структура CSV файла

CSV файл содержит следующие колонки:

- `infinitive` - инфинитив глагола
- `past_simple` - форма прошедшего времени
- `past_participle` - причастие прошедшего времени
- `transcription_inf` - транскрипция инфинитива
- `transcription_ps` - транскрипция past simple
- `transcription_pp` - транскрипция past participle
- `translation` - перевод на русский язык
- `example_en` - пример предложения на английском
- `example_ru` - пример предложения на русском

## Типы карточек

Генерируется 5 типов карточек для каждого глагола:

1. **Infinitive → Past Simple**: даны инфинитив и past participle, нужно ввести past simple
2. **Past Simple + Past Participle → Infinitive**: даны две формы, нужно найти инфинитив
3. **Infinitive + Past Simple → Past Participle**: даны две формы, нужно найти past participle
4. **English → Russian**: дан английский глагол, нужно ввести перевод
5. **Russian → English**: дан русский перевод, нужно ввести английский глагол

## Возможности карточек

- **Интерактивные поля ввода** с проверкой правильности ответов
- **Кнопки озвучивания** для всех английских форм глагола
- **Транскрипция** для правильного произношения
- **Примеры предложений** на английском и русском языках
- **Визуальная обратная связь** (зеленые/красные поля при правильных/неправильных ответах)

## Использование

### Предварительные требования

```bash
pip install genanki
```

### Генерация колоды

```bash
python generate_verbs_deck.py verbs.csv
```

Дополнительные параметры:

```bash
python generate_verbs_deck.py verbs.csv -o my_verbs.apkg -n "My Irregular Verbs"
```

Параметры:

- `-o, --output` - имя выходного файла (по умолчанию: irregular_verbs.apkg)
- `-n, --name` - название колоды (по умолчанию: "Irregular English Verbs")

### Импорт в Anki

1. Откройте Anki
2. Выберите "File" → "Import"
3. Выберите созданный файл `irregular_verbs.apkg`
4. Колода будет добавлена в вашу коллекцию
